# 个人品牌网站项目规划

## 一、项目概述

### 1.1 项目定位
打造个人AI副业品牌网站，树立专业技术形象，沉淀内容资产，建立与目标受众（对AI和副业感兴趣的群体）的连接渠道。

### 1.2 核心目标
- **个人IP塑造**：展示程序员的AI副业之路，建立专业可信形象
- **内容沉淀**：系统化整理和分享AI智能体相关内容
- **产品展示**：展示和销售AI智能体产品（工作流模板、定制智能体等）
- **流量获取**：通过SEO和内容营销吸引目标用户

---

## 二、网站架构设计

### 2.1 页面结构

```
网站首页
├── 关于我 (About)
│   ├── 个人背景介绍
│   ├── AI副业经历
│   ├── 技能栈展示
│   └── 联系方式/社交媒体
│
├── 博客/文章 (Blog)
│   ├── 文章列表（按分类/标签）
│   ├── 文章详情页
│   └── 搜索功能
│
├── AI智能体产品 (Products)
│   ├── 产品列表
│   ├── 产品详情页
│   └── 产品分类（工作流模板/定制智能体/插件等）
│
└── 资源/工具 (Resources) [可选扩展]
    ├── 免费工具
    └── 资源下载
```

### 2.2 核心功能模块

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| 首页 | 个人品牌入口、最新文章/产品展示 | P0 |
| 关于我 | 个人IP故事、联系方式 | P0 |
| 博客系统 | 文章发布、分类、标签、搜索 | P0 |
| 产品展示 | 产品列表、详情、购买/咨询入口 | P0 |
| SEO优化 | Meta标签、结构化数据、Sitemap | P0 |
| 响应式设计 | 移动端/平板/桌面端适配 | P0 |
| 订阅功能 | 邮箱订阅获取最新内容 | P1 |
| 评论系统 | 文章互动（可延后实现） | P2 |

## 三、技术选型方案

### 3.1 推荐技术栈

| 层级 | 技术选型 | 理由 |
|------|---------|------|
| **框架** | Astro | 轻量快速、内容为王、SEO友好、支持MDX |
| **样式** | Tailwind CSS | 快速开发、响应式、易定制紫色主题 |
| **内容** | MDX (Markdown) | 程序员友好、易维护、无需数据库 |
| **部署** | Vercel/Netlify | 免费、自动部署、全球CDN |
| **域名** | 待注册 | 推荐 `.com` 或 `.ai` 域名 |
| **分析** | Google Analytics / Umami | 隐私友好的数据分析 |

### 3.2 备选方案

| 方案 | 适用场景 | 特点 |
|------|---------|------|
| **Next.js** | 如果需要复杂交互 | 功能强大、生态丰富、学习曲线略高 |
| **WordPress** | 如果不熟悉代码 | 生态成熟、插件丰富、但较重 |
| **Notion + Super** | 快速上线 | 极简、但定制性有限 |

### 3.3 SEO优化方案

- **技术SEO**：Meta标签优化、结构化数据（JSON-LD）、Sitemap自动生成
- **内容SEO**：文章分类策略、标签系统、内部链接优化
- **性能SEO**：图片优化、静态生成、CDN加速
- **工具集成**：Google Search Console、SEO分析插件
